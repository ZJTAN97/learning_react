# :snowflake: Custom Webpack React Setup :snowflake:

- A custom react app set up using webpack for reusability across projects done by me.
- Provides the bare minimum settings and boiler plate to get started with development in React.
- Styling default is with SASS
- A custom loader for webpack is written to generate SCSS typings upon compilation.
- Default testing runner is Jest and library used is React Testing Library

<br>

# :snowflake: Tech Stack :snowflake:

- React 18
- TypeScript 4.9
- Webpack 5

<br>

# :snowflake: Usage :snowflake:

```

yarn install
yarn watch // starts dev server at port 3000

```
